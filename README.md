# Git 練習專案

III MFEE20 的 git 練習專案

| Windows | MacOS/Linux | 說明                          |
| ------- | ----------- | ----------------------------- |
| cd      | cd          | 切換目錄 change directory     |
| cd      | pwd         | 顯示目前所在路徑              |
| dir     | ls          | 列出目前檔案夾的檔案列表 list |
| mkdir   | mkdir       | 建立新的檔案夾 make dir       |
| copy    | cp          | 複製檔案 copy                 |
| move    | mv          | 移動檔案 move                 |
| del     | rm          | 刪除檔案 remove               |
| cls     | clear       | 清除畫面上的內容              |
| cat     | cat         | 印出檔案內容                  |
